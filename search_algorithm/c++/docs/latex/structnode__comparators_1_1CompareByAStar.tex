\subsubsection*{template$<$typename TState, typename TAction$>$\newline
struct node\+\_\+comparators\+::\+Compare\+By\+AStar$<$ TState, TAction $>$}

Comparator for A$\ast$ search algorithm. 

Orders nodes by their evaluation function f(n) = g(n) + h(n), where\+:
\begin{DoxyItemize}
\item g(n) is the path cost from start to node n
\item h(n) is the heuristic estimate from node n to goal
\end{DoxyItemize}

Creates a min-\/heap where nodes with lower f-\/values have higher priority. This ensures A$\ast$ always expands the most promising node first.

\begin{DoxyNote}{Note}
Need to provide a \mbox{\hyperlink{classProblem}{Problem}} instance in construction to access the heuristic function
\end{DoxyNote}
