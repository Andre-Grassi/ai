\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TState, typename TAction$>$\newline
class Problem$<$ TState, TAction $>$}

Abstract base class for search problems. 

This class defines the interface that all search problems must implement. It provides the basic components needed for search algorithms\+:
\begin{DoxyItemize}
\item Initial state
\item Goal testing
\item Action generation
\item State transitions
\item Action costs
\item Heuristic evaluation
\end{DoxyItemize}
