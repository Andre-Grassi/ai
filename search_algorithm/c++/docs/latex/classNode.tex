\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TState, typename TAction$>$\newline
class Node$<$ TState, TAction $>$}

Represents a node in a search tree. 

A node contains a state, a pointer to its parent node, the action that led to this state, and the cumulative path cost from the initial state. Nodes are used to build search trees in various AI search algorithms.
