\doxysubsection{Detailed Description}
Comparator that orders nodes by their path cost. 

Orders nodes by path cost in ascending order, creating a min-\/heap where nodes with lower path costs have higher priority. This is used by uniform cost search to always expand the lowest-\/cost frontier node first.

\begin{DoxyNote}{Note}
Returns true if \textquotesingle{}a\textquotesingle{} should have lower priority than \textquotesingle{}b\textquotesingle{} in a priority queue 
\end{DoxyNote}
