Detailed Description

template<typename TState, typename TAction>
class Problem< TState, TAction >

Abstract base class for search problems.

This class defines the interface that all search problems must implement. It provides the basic components needed for search algorithms:

  • Initial state
  • Goal testing
  • Action generation
  • State transitions
  • Action costs
  • Heuristic evaluation
Template Parameters
TStateType representing a state in the problem space
TActionType representing actions that can be applied to states
