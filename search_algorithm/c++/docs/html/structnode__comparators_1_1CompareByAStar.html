Detailed Description

template<typename TState, typename TAction>
struct node_comparators::CompareByAStar< TState, TAction >

Comparator for A* search algorithm.

Orders nodes by their evaluation function f(n) = g(n) + h(n), where:

  • g(n) is the path cost from start to node n
  • h(n) is the heuristic estimate from node n to goal

Creates a min-heap where nodes with lower f-values have higher priority. This ensures A* always expands the most promising node first.

Note
Need to provide a Problem instance in construction to access the heuristic function
